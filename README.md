# rs2glsl

## Usage

### Base shader build
For shadertoy

```
$ rs2glsl examples/basic.rs examples/adapters/shadertoy.rs
```

For Kodelife

```
$ rs2glsl examples/basic.rs examples/adapters/kodelife.rs
```

### Koyuki shader build
For shadertoy

```
$ rs2glsl examples/koyuki.rs examples/adapters/shadertoy.rs examples/std.rs
```

For Kodelife

```
$ rs2glsl examples/koyuki.rs examples/adapters/kodelife.rs examples/std.rs
```
